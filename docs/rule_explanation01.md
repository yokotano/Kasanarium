ゲームで使用するもの
--------------------
### バイオームカード
テラリウム内の環境を表現します
カードは3x3のマスに区切られており、各マスにはそれぞれ4種類のバイオームのうちいずれか1つが割り振られます
プレイヤーは箱庭に手札からバイオームカードを置いて箱庭を広げることができます
箱庭はすべてのプレイヤーで共有されます

バイオームには以下の種類があります
- 草
- 水
- 岩
- 土

### いきもの
テラリウム内で飼育する生物を表現します

各 _いきもの_ には配置するために満たさなけばならない条件が設定されています
配置するための条件はミッションカードに記載されています
プレイの進行により条件を満たさなくなっても配置済みの _いきもの_ が取り除かれることはありません

_いきもの_ は以下の条件を満たしたマスに配置することができます
- そのターンに自身が置いたバイオームカード上にある
- 配置するための条件となる地形に含まれる
- ほかの _いきもの_ が配置されていない

### ミッションカード
達成すると得点が得られるミッションと得られる得点が書かれたカードです
ゲームの終了時には獲得したミッションカードの得点の合計で勝敗が決まります
ミッションはおおきく以下の種類に分けられます

- いきものミッション: 箱庭に特定の _いきもの_ を配置することで得点が得られます
- バイオームミッション: 箱庭に特定のバイオームの組み合わせ(地形)を作成することで得点が得られます。自身がそのターンに置いたバイオームカードが地形に含まれる場合に限り達成可能です

その他の用語
------------
### 箱庭
バイオームカードが置かれる場のこと
ゲームの終了時にはプレイヤー同士がゲームを通じて築いた箱庭が完成します
ぜひ眺めたり写真を撮ったりしてその風景を楽しんでみてください

### 地形
マスの固まりのこと
ミッションカードでは"特定のマス (数値)"のように達成するための条件が書かれています
指定されたマスがその数値以上隣接している場合、その条件が達成できます

ミッションによっては複数の種類のマスを要求したり、マスの数だけでなく形も指定されるものもあります
