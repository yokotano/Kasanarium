Kasanarium (重ねる + -arium)
============================
Kasanariumはカードを重ねてテラリウムをつくる、2人用のボードゲームです  
生育に適した環境を整えていきものを配置したり地形を整えたりしながら2人で1つの箱庭をつくりあげ、より多くの得点を獲得したプレイヤーの勝利となります

ゲームの終了時にはプレイヤー同士がゲームを通じて築いた箱庭が完成します  
ぜひ眺めたり写真を撮ったりしてその風景を楽しんでください

ゲームで使用するもの
--------------------
### バイオームカード
テラリウム内の環境を表現します  
カードは3x3のマスに区切られており、各マスには以下の4種類のバイオームのうちいずれか1つが割り振られます
- 草
- 水
- 岩
- 土

プレイヤーは手札からバイオームカードを重ねて箱庭を広げることができます  
箱庭はすべてのプレイヤーで共有されます

### ミッションカード
達成すると得点が得られるミッションと得られる得点が書かれたカードです  
ゲームの終了時には獲得したミッションカードの得点の合計で勝敗が決まります  
ミッションには"バイオームミッション"と"いきものミッション"の2種類があります

### いきもの
テラリウム内で飼育する生物を表現するトークンです

各 _いきもの_ には配置するために満たさなけばならない条件が設定されています  
配置するための条件はミッションカードに記載されています  
プレイの進行により条件を満たさなくなっても配置済みの _いきもの_ が取り除かれることはありません

ゲームの準備
------------
ゲームを開始する際にバイオームカードとミッションカードをそれぞれシャッフルします  
ミッションカードとバイオームカードは山札としてテーブルの端に置いてください  
(このゲームではテーブルを広く使用します。山札はのちのプレイの妨げにならないように適宜移動させてください)

各プレイヤーに5枚ずつバイオームカードを山札の上から引き裏向きで配ります  
バイオームカードはほかのプレイヤーに見せないようにしてください  
続いて、ミッションカードを3枚山札から引き、表向きにしてすべてのプレイヤーが確認できる位置に配置します

最後に、先行・後攻を決めます  
じゃんけんやコイントスなど好きな方法で決めてください

ゲームの進め方
--------------
ゲームは各プレイヤーが交互にターンを行うことで進行します  
ターン中、終了条件が満たされた場合、そのターンでゲームが終了します  
その際、達成できたボーナスがあれば達成したプレイヤーはその点数を獲得します  
最終的に最も多くの得点を獲得したプレイヤーが勝者となります

各ターンは以下のように進行します
1. ミッションカードが3枚より少なくなっている場合、3枚になるまで山札から引いて補充します
2. 手札からバイオームカードを1枚箱庭に重ねます
3. 公開されているミッションカードから条件を満たしているものがあれば宣言して取得します
4. 手札が5枚になるようにバイオームカードを山札から手札に補充します

### 終了条件
以下のいずれかの条件が満たされた場合、そのターンでゲームが終了します
- すべての _いきもの_ が箱庭に配置される
- ターン終了時にバイオームカードが山札から引けない

### ボーナス
ゲームの終了時に以下の条件を満たしている場合、そのプレイヤーは追加で得点を獲得します  
ボーナスは1種類につき1度のみ獲得できます
- 食物連鎖(陸): 5点 / バッタ、カエル、トカゲのミッションカードをそれぞれ1枚以上獲得している
- 食物連鎖(水): 5点 / メダカ、ザリガニ、カメのミッションカードをそれぞれ1枚以上獲得している
- おんぶバッタ: 3点 / バッタのミッションカードを2枚以上獲得している
