Kasanarium (重ねる + -arium)
============================
Kasanariumはカードを重ねてテラリウムをつくる、2人用のボードゲームです
_いきもの_ の生育に適した箱庭をつくり、より多くの得点を獲得したプレイヤーの勝利となります

ゲームの進め方
--------------
ゲームを開始する際にバイオームカードとミッションカードをそれぞれシャッフルしてください

ゲームは以下のように進行します
1. 好きな方法で先攻・後攻を決めます
2. 各プレイヤーは5枚ずつバイオームカードを山札から引いて手札にします
3. ミッションカードを3枚山札から引き、表向きにしてすべてのプレイヤーが確認できる位置に配置します
4. ゲームの終了条件が満たされるまでターンを交互に行います
5. 各プレイヤーは達成できたボーナスがあればその点数を獲得します
6. ゲームの終了時に最も多くの得点を獲得したプレイヤーが勝者となります

各ターンは以下のように進行します
1. ミッションカードが3枚より少なくなっている場合、3枚になるまで山札から引いて補充します
2. 手札からバイオームカードを箱庭に置きます
3. 公開されているミッションカードのうち、条件を満たしたものがあればそれらを宣言できます
4. 宣言したミッションカードの条件達成が確認できたら、そのミッションカードを獲得します
5. 手札が5枚になるようにバイオームカードを山札から手札に補充します

バイオームカードの置き方
------------------
バイオームカードの設置には以下の制約があります
- 箱庭に1枚もバイオームカードがない場合を除き、すでに箱庭に置かれたバイオームカードの上にいずれかに1マス以上重ねて置く必要があります
- 重ねる際、バイオームカードの向きは正位置か180度回転した向きでのみ置くことができます
- 9マスすべてのマスが見えているバイオームカードの上に、ぴったり(9マスすべてが)重なるように置くことはできません
- _いきもの_ が配置されているマスに重ねて置くことはできません

ゲームの終了条件
----------------
以下のいずれかの条件が満たされた場合、そのターンでゲームが終了します
- すべてのいきものが配置される
- ターン終了時にバイオームカードが山札から引けない

ボーナス
--------
ゲームの終了時に以下の条件を満たしている場合、そのプレイヤーは追加で得点を獲得します
ボーナスは1種類につき1度のみ獲得できます
- 食物連鎖(陸): 5点 / バッタ、カエル、トカゲのミッションカードをそれぞれ1枚以上獲得している
- 食物連鎖(水): 5点 / メダカ、ザリガニ、カメのミッションカードをそれぞれ1枚以上獲得している
- おんぶバッタ: 3点 / バッタのミッションカードを2枚以上獲得している
